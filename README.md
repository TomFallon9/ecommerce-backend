## Description
App built as a backend to an E-Commerce website. Express.js was used for the server while  MySQL and  Sequelize as the ORM to run the database models and queries.
The database includes tables for products, categories, tags, and product tags. RESTful API routes are used to make requests and updates from the database then joined through Sequelize

## Installation

- Clone the reo to your local machine: 
  - `git clone git@github.com:tomfallon9/ecommerce-backend.git`
- Navigate to the root directory and run: 
  - `npm install`
- To start the server, in the command line run: 
  - `npm start`
- To add seeds run:
  - `npm run seed` 
- Enter DB user name and password of your own database. If cloning or pushing to Github make sure to add a dotenv .env folder with your credentials. Make sure to also add .env to your .gitignore
